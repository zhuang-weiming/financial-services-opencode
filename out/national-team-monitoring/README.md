# 国家队全口径资金监测方案

> **创建:** 2026-07-22 | **状态:** 初版规划（待执行）
> **来源:** 框架审计 #1—"2026年国家队是净卖出者，需要数据决定安全边际"
> **目标:** 通过公开数据，追踪国家队（汇金/社保/保险/SFISF）的真实仓位变化，判断当前 regime 是净买入还是净卖出

## 三、核心监控指标（简化版）

为回答「国家机器是净买入还是净卖出」，聚焦以下可获取的高频指标：

### 3.1 日频指标（最高优先级）

```
510300（华泰柏瑞沪深300ETF）每日份额变化
   ↓ 连续3日净赎回 > 1亿份 → 可能国家队在卖出
   ↓ 连续3日净申购 > 1亿份 → 可能国家队在买入

辅助验证：
  510050（上证50ETF）、510310（易方达沪深300ETF）、159919（嘉实沪深300ETF）
  三只同步变化 → 信号可靠
  单只变化 → 可能是散户/机构行为
```

**判断标准：**

| ETF份额变化(周) | 含义 |
|:---------------|:-----|
| 四大ETF合计周净申购 > 50亿份 | 🟢 国家队可能是净买入（2024年模式） |
| 四大ETF合计周净赎回 > 30亿份 | 🔴 国家队可能是净卖出（2026H1模式） |
| 变化 < ±20亿份 | 🟡 中性—国家队未干预 |

### 3.2 月频指标

```
保险资金权益比例（金融监管总局官网数据，手动查询）
   > 12% → 🟢 保险在增配权益
   < 10% → 🔴 保险在减配

两融余额趋势
   持续上升 → 🟢 市场自我修复
   持续下降 → 🔴 市场风险偏好不足

北向资金月度净流入
   月度 > 500亿 → 🟢 外资回流
   月度 < -300亿 → 🔴 外资出逃
```

### 3.3 季度指标

```
汇金ETF季末持仓（基金季报，滞后1-2月）
社保基金前十大持仓变化
保险资金AUM + 权益配置比例
```

## 五、关键判断：2026 年 7 月现状

基于 FactSet 实时数据 + BT-004 分析：

| 指标 | 值 | regime 判断 |
|:-----|:---|:------------|
| 汇金 ETF 2024H1 | 净买入 ~+5,000 亿 | 🟢 净买入 |
| 汇金 ETF 2026H1 | 净卖出 ~-1.6 万亿（彭博估算） | 🔴 净卖出 |
| **FactSet 510050份额** | 566亿→64亿（-89%） | **🔴 确认净卖出** |
| **FactSet 510310份额** | 659亿→116亿（-82%） | **🔴 确认净卖出** |
| **FactSet 159919份额** | 408亿→69亿（-83%） | **🔴 确认净卖出** |
| 保险权益比例 | 10.1%（同比+2.6ppt） | 🟢 缓慢增配 |
| SFISF 使用率 | ~20%（1,050/5,000亿） | 🟡 中性（半废弃） |
| **综合判断** | **2026H1 = 净卖出 regime（数据确认）** | **🔴** |

> **对安全边际的影响：** FactSet 数据证实了 2026Q2 国家队系统性赎回四大沪深300ETF。份额减少不是散户行为（AUM 保持稳定甚至增长），而是 AP 大宗赎回——最可能的操盘方就是汇金。当前 regime 下，**"国家机器下跌时会买入"的假设不成立**。沪指 3000 点"政策底"可能已下移至 2800-3000 区间。框架需将"安全边际"预期下调~10%。
