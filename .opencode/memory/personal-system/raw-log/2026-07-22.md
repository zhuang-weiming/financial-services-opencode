## 2026-07-22 21:30 [TAG: BACKTEST] [BT-003 — 券商PB vs 成交量回测]
- **触发:** 用户提出假设"成交量放大→券商PB修复→股价上涨2-3倍"，要求用历史数据验证
- **方法:** 回测2010-2026年A股市场日均成交额 vs 券商板块（中信/华泰/东财/银河/招证/广发/国君/海通）平均PB，月度频率Pearson相关+领先滞后+分时期分析
- **数据源:** baostock + eastmoney / AKShare / 中证全指证券公司指数 (399975)
- **关键结果:**
  - **全样本Pearson r = -0.13**（负相关！p=0.15不显著）
  - **R² = 0.007**（成交量解释不了PB任何变化）
  - 2015年月均成交7,530亿→券商PB 5.05x；2026年月均成交31,117亿→券商PB 1.54x（量增4倍，PB跌70%）

- **归因:** 用户假设的A→B→C链中断在A→B（成交量→券商收入）。佣金率塌陷+业务结构变化使量价脱钩。
