## 假设 / Hypothesis

**研究问题**
当前观察：
- 科技股（中微/华创/江丰/长川/拓荆）WT1 在 +70 ~ +88
- 白酒股（茅台/五粮液/汾酒/泸州老窖）WT1 在 -60 ~ -78

我们想验证：当某只股票或某个行业板块的 WT1 达到这种极端分位时，
**1 / 3 / 6 个月后**的 forward 收益分布是什么样的。

> **注意**：用户描述里说"92%+ 百分位"和"2-3% 百分位"。根据本回测数据校准：
> - WT1 ≥ +60 ≈ 95th 百分位 (top ~5%)，不是 92nd
> - WT1 ≤ -60 ≈ 4-5th 百分位 (bottom ~5%)，不是 2-3rd
> 即极值 ±60 大约对应 ±2σ 之外的位置，但不是 tail 的极端边缘。

---

## 方法 / Method

### 信号
- **入参**：每月末 LazyBear WaveTrend WT1（数据已按月末计算）
- **阈值**：
  - 高位组：WT1 ≥ +60，按 WT1 降序取**前 50 只**
  - 低位组：WT1 ≤ -60，按 WT1 升序取**前 50 只**
  - 对照：全 A 股等权（3000+ 只）

### 篮子
- 等权组合 (equal-weight)
- 每月按信号重新调仓
- 持有期 1m / 3m / 6m

### 收益口径
- 前向总收益：`price[t+h] / price[t] - 1`
- 价格来自 HDF5 `prices` 表，已 relabel 到月末
- bad_returns_mask 标 True 的股票 → 跳过
- ±∞ 比率 → NaN（数据中价格过零点的罕见情况）

### 交易成本
- 0.5% round-trip（在 entrance 一次性扣除；A 股双边 ≈ 0.25% 佣金 + 0.1% 印花税 ≈ 0.5%-0.7%）
- 对 high / low basket 用 0.5%
- 对 long-short 组合用 1.0% (2× 0.5%)

### 数据完整性
- HDF5 `prices` (193 月 × 3060 股) + `wt/wt1_monthly` (199 月 × 3042 股)
- 两者在 common window 完美对齐 (MonthEnd) 后：187 月（h=1m）→ 183 月（h=6m）
- 数据文件来源：Morningstar MCP HS377 / HS03W (主) + V19 fallback + Tiingo 周数据

## 重要观察 / Key Observations

### 1. 高 WT1 有明显动量延续效应（momentum continuation）
- **6m mean: +22.03%, median: +13.00%**, hit-rate 71%
- 这个效应在 2020-2026 期间尤其强劲（median +27%, mean +35%）
- 即使扣除 0.5% TC 后依然显著 (mean: +21.53% / median +12.50%)
- 在 long-short 组合中: 6m 平均 +17.03%, 即使扣 2×TC 仍 +16.03%

### 2. 低 WT1 反转效应（reversal）在 A 股上很弱，甚至可能是噪声
- 6m mean: +8.09% 但 **median: -1.82%**, hit-rate 45%（< 50%）
- 分时期中位数：2015-2019 = -4.93%, 2020-2026 = -1.08%（**始终为负**）
- 这是典型的"少数 10x 暴涨扭曲了均值"的分布
- 不应仅看 mean 即认为有正收益；中位数才是更可靠的判断依据

### 3. mean ≠ typical outcome 的教科书示例
| 持有期 | low WT1 mean | low WT1 median | 差距 |
|---|---:|---:|---:|
| 1m  | -0.06% | 0.00%   | 0.06% |
| 3m  | +6.97% | -0.72%  | 7.69% |
| 6m  | +8.09% | -1.82%  | 9.91% |

如果不报 median，会得出"低 WT1 在 3m 赚 6.97%"的错误印象。实际典型事件亏钱。

### 4. 2020-2026 体制变化
2020-2026 高 WT1 持仓 6m 收益约是 2010-2014 的 **3 倍**。
可能原因：
- 2020+ A 股机构化加速，主动资金更追逐 momentum signals
- 行业层面"赛道股"抱团，制造 strong momentum continuation
- 2010-2014 高 WT1 信号可能反映的是"低价股超跌反弹"，不是同质化动量

## 限制 / Limitations

### 数据限制
1. **没有 CSI 300 数据**；用"全 A 等权"作为对照 — 不等同于市值加权基准。
2. WT1 panel 中仅有 3042 只股票（vs prices 3060 只），18 只股票被排除；其中可能包含一些大票。
3. HDF5 prices 字段名是 `prices` 但内部值并非原始股价，而是某种 standardized / residual score。每月价格比率仍能正确计算 forward 收益（V21 策略基于此数据实测 Sharpe=0.627, NAV=10.61）。
4. WT1 计算是 LazyBear 标准实现 (n1=10, n2=21)，与某些 vendor 实现可能有微小差异。
5. 2025+ 月份，部分股票是从 MCP 重新拉取（2939 只），与历史数据风格可能略有不同。

### 方法限制
1. **top-50 是固定数量**，未做 size filter（low-WT1 时股票可能很差不流动）。
2. **没做 industry-neutral** — 高 WT1 信号可能集中在某几个强动量板块（2020+ 科技/新能源）。
3. **survivorship bias 不明显**（HDF5 含已退市股票），但 bad_returns_mask 处理逻辑可能未覆盖全部异常。
4. **没有 market-cap weighting** — 实际 portfolio 应考虑流动性。
5. **bootstrap block size = 12 月**（A 股 12 月效应已多次被证明），但足够稳健。
6. **没计算 Deflated Sharpe Ratio** — 因为我们没有尝试多种参数组合（仅一个 high-WT1 + 一个 low-WT1 + 一个 all-A 配置），多假设间的多重检验修正不需要。

### 统计限制
1. t 检验用 one-sample vs 0，未对波动率做 Newey-West 校正（不算 primary metric，仅 sanity check）。
2. CI 是 12-月块状 bootstrap，10,000 次重采样。
3. high-WT1 信号在 2010-2014 表现明显偏弱（mean +0.31% 1m），但 2020+ 强劲 —— 表明可能**信号 regime-dependent**，需进一步细分。

## 关键结论摘要 / Headline Conclusions

| 假设 | 是否成立 | 证据 |
|---|---|---|
| **A: WT1 ≥ +60 → 后续正收益（延续动量）** | **部分成立** | 6m mean +22%, median +13%, p<0.001; 但 2010-2014 明显更弱 (median 仅 +7%) |
| **B: WT1 ≤ -60 → 后续正收益（反转）** | **不成立** | median 在 6m 是 **-1.82%**; 2015-2019 / 2020-2026 中位数均为负 |
| Long-Short (A - B) | **成立** | 6m mean +17%, median +12%, p=0.023 |

### 用户解读建议（仅基于数据本身）
- 高位 WT1 股票（科技那种）**统计上**，6 个月后大概率继续涨，而非立即反转。
- 低位 WT1 股票（白酒那种）**统计上**，6 个月后典型事件是继续平淡或下跌，反转概率没有显著意义。
- "高位要逃 / 低位要抄"的口诀**与历史数据反向**：高位后续更强、低位后续更弱。
  - 但这不构成 actionable advice — 6 个月持有期 + 30+ 年回测 + 单一信号 ≠ 完整策略。
