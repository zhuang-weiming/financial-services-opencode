# 券商板块观察指标体系

> 创建: 2026-07-22
> 基于: BT-003回测（"成交量驱动PB修复"假设被证伪后的修正框架）
> 状态: 激活 — 定期检查

---

## 核心认知变更

BT-003回测明确显示：**"成交量放大→券商PB修复→股价2-3倍"的原始假设在2015年后已失效。**

修正后的驱动力框架：

```
过时模型:  成交量 ↗ → 佣金收入 ↗ → PB修复 ↗ → 股价 ↗
            ↓
修正模型:  [市场beta(沪深300相关r>0.93)] + [利率周期] + [监管政策] + [居民存款搬家] 
            → 券商收入/利润改变 → PB修复 → 股价变化
成交量 仅作为 辅助确认信号（不是驱动因素）
```

## 观察指标分级体系

### 第一层：宏观先行信号（提前6-12个月）

| 指标 | 当前值 | 触发值 | 逻辑 | 检查频率 |
|:---|---|:---:|:---|:---:|
| **沪深300趋势** | 待查 | 连续3月站上年线 | 券商Beta>0.93，等大盘先动 | 月度 |
| **10年期国债收益率** | 待查 | 下行突破2.0% | 利率下行降低券商融资成本+提升估值 | 月度 |
| **居民存款搬家速度** | 上半年少增3.19万亿 | 季度少增>1.5万亿 | 资金向非银迁移→部分入市 | 季度（央行数据） |
| **监管政策信号** | — | IPO注册制/做市商/降税 | 政策驱动最不可预测但弹性最大 | 事件驱动 |

### 第二层：券商基本面确认（同步信号）

| 指标 | 观察点 | 关键值 |
|:---|---|:---:|
| **龙头券商PB** | 中信证券当前~1.5x | 突破1.8x → 确认修复启动；突破2.5x → 过热 |
| **券商板块成交占比** | 当前~5-8% | 升至12-15% → 板块主线确认 |
| **两融余额** | 当前~1.8万亿 | 站上2.5万亿 → 散户杠杆加速入场 |
| **券商季度营收增速** | 关注经纪+投行+自营三条线 | 连续2季正增长 → 基本面拐点确认 |

### 第三层：退出信号（预计2026下半年-2027中）

| 信号 | 触发动作 | 优先级 |
|:---|---|:---:|
| **美债CDS从当前~10bp飙至50bp+** | ⚠️ **全面清仓**（早于2027.7.1） | 最高 |
| **券商板块PB > 2.5x** | 减仓50% | 高 |
| **AI板块季度增速降至50%以下** | 开始减仓 | 高 |
| **连续两月成交量从峰值回落20%+** | 减仓30% | 中 |
| **居民贷款连续2季度同比少增** | 减仓30%（宏观确认） | 中 |
| **2027年7月1日到达** | **全面清仓**（纪律执行） | 最高 |

## 月度检查清单

每月1日执行以下检查：

```
□ 沪深300月线是否站上/跌破关键位（年线/前高）
□ 日均成交量 万亿/1.5万亿/2万亿
□ 中信证券PB当前值
□ 两融余额趋势
□ 美债CDS（关注是否>30bp）
□ 北向资金月度净流入/流出
□ AI板块龙头（中际旭创）季度增速
```

## 退出流程（按触发顺序）

```
一级预警（美债CDS>50bp 或 2027年4月）:
  → 减至50%仓位
  → 准备全面清仓清单

二级预警（2027年6月）:
  → 减至20%仓位
  → 仅保留现金等价物

最终执行（2027年7月1日）:
  → 0%仓位
  → 写入raw-log: 清仓完成
```

## 免责声明

此观察体系基于历史数据的量化规律总结。历史规律可能因市场结构变化而失效（如BT-003显示的成交量与PB关系的结构性断裂）。所有信号均需结合当时的市场环境综合判断。
