# Data Priority Instructions

## Critical: Data Source Hierarchy

**ALWAYS follow this data source hierarchy:**

1. **FIRST: Morningstar MCP** - Use for investment research, analyst estimates, fund data, and equity analysis
2. **SECOND: FactSet MCP** - Use for institutional financial data, earnings, valuations, and market intelligence
3. **THIRD: DDG Search (Internet)** - Only use for general web search when MCP sources are unavailable or for supplementary information
4. **NEVER use web search as a primary data source** - it lacks the accuracy, audit trails, and reliability required for institutional-grade analysis

## Why This Matters

- MCP sources provide verified, institutional-grade data with proper citations
- Morningstar MCP offers comprehensive equity research, fund analysis, and estimate data
- FactSet MCP provides detailed financial statements, trading data, and valuation metrics
- Web search results can be outdated, inaccurate, or unreliable for financial analysis

## Skill Activation

When invoking any financial analysis skill, the skill will automatically activate based on the task trigger conditions. Always ensure the data source priority is followed when gathering financial information.
