## 三大报表与主要指标（Financial Statements + GMAININDICATOR）

工具：`get_financial_statements`（`src.tools.financial_statements_tool.FinancialStatementsTool`）
描述：取单只股票的资产负债表 / 利润表 / 现金流量表 / 主要指标，按 symbol 后缀选上游：
- **美股（.US）** —— SEC EDGAR companyfacts：ticker 解析到 CIK，再读取 XBRL us-gaap concepts，聚合成逐期扁平行。
- **A 股（.SH / .SZ / .BJ）与港股（.HK）** —— 东财 datacenter 报表接口；三大报表来自分市场财报数据集，`indicators` 读主要指标数据集（GMAININDICATOR）。
报表以"最新在前"的扁平逐期行返回。
市场：美股（SEC EDGAR）、A 股/港股（东财）。
限速：SEC 经 `sec` bucket，东财经 `eastmoney` bucket，均 per-host 节流。

<br>

**端点**

| 市场 | 端点 |
| ---- | ---- |
| 美股 | `https://data.sec.gov/api/xbrl/companyfacts/CIK##########.json` |
| A 股/港股 | `https://datacenter.eastmoney.com/securities/api/data/v1/get` |

**东财 reportName（market_group → statement → reportName）**

statement | a（0/1） | hk（116）
---- | ---- | ----
balance | `RPT_F10_FINANCE_GBALANCE` | `RPT_HKF10_FN_BALANCE`
income | `RPT_F10_FINANCE_GINCOME` | `RPT_HKF10_FN_INCOME`
cashflow | `RPT_F10_FINANCE_GCASHFLOW` | `RPT_HKF10_FN_CASHFLOW`
indicators | `RPT_F10_FINANCE_MAINFINADATA` | `RPT_HKF10_FN_GMAININDICATOR`

**SEC concepts（美股）**：按 statement 读取一组常见 us-gaap concepts，例如 income 包含 `Revenues` / `RevenueFromContractWithCustomerExcludingAssessedTax` / `NetIncomeLoss` / EPS，balance 包含 `Assets` / `Liabilities` / `StockholdersEquity`，cashflow 包含经营/投资/融资现金流。缺失 concept 会被跳过，不使整个工具失败。

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
code | str | Y | 单只带后缀 symbol，如 `600519.SH` / `000001.SZ` / `AAPL.US` / `00700.HK`。
statement | str | N | `balance` / `income` / `cashflow` / `indicators`（默认 `indicators`）。
period | str | N | `annual`（年报）或 `quarter`（季报），默认 `annual`。

<br>

**端点查询参数**

SEC：先查 ticker->CIK，再取 companyfacts；`annual` 保留 `fp=FY` 或 `form=10-K` 的点，`quarter` 返回完整逐期点。
东财：`reportName=<上表>`、`columns=ALL`、A 股 `filter=(SECUCODE="<code>")`，港股 `filter=(SECURITY_CODE="<bare>")`，`sortColumns=REPORT_DATE`、`sortTypes=-1`、`pageNumber=1`、`pageSize=40`、`source=F10`、`client=PC`。

<br>

**返回字段**

报表逐期行为扁平 `字段名 → 值`。A 股/港股保留东财原始字段；美股逐期行包含 `REPORT_DATE` / `FISCAL_YEAR` / `FISCAL_PERIOD` / `FORM` / `ACCESSION`，以及 us-gaap concept 名称和值，`_units` 记录每个 concept 的单位。防爆上限：每只股票最多 40 期，每期最多 200 个字段。

信封：`{"ok": true, "market": "a_share"|"us"|"hk", "source": "sec_edgar"|"eastmoney", "statement": <statement>, "period": <period>, "data": {code: {"periods": [...]} | {"error": ...}}}`。单只 symbol 失败以 per-symbol `error` 上报，不抛异常。

<br>

**调用范例**

```python
from src.tools.financial_statements_tool import FinancialStatementsTool

# 美股利润表（年报）
print(FinancialStatementsTool().execute(code="AAPL.US", statement="income", period="annual"))
# 港股主要指标（GMAININDICATOR）
print(FinancialStatementsTool().execute(code="00700.HK", statement="indicators"))
# A 股资产负债表（东财）
print(FinancialStatementsTool().execute(code="600519.SH", statement="balance"))
```
