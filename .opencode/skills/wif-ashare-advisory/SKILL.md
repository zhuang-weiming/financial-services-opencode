# WIF 中国A股基金配置（Wealth Investment Framework v2.7）

## 概述

WIF v2.7 是基于 PMI+M2 宏观数据的 A 股多资产配置框架。通过**宏观景气指数（MCI）** 确定市场象限（Q1/Q2/Q3），叠加 **MA60 趋势覆盖**增强趋势暴露，配合 **EMERGENCY 三级保护** 应对崩溃行情。

## 核心方法论

### 信号流程图

```
每月末 PMI 发布 ──┐
每月中 M2 发布  ──┤──→ MCI（宏观景气指数）──→ 象限（Q1/Q2/Q3）
                  │                             │
每日 HS300 收盘 ──┘                          ↓趋势覆盖?
                                         Trend>7% → Q1 强制
                                         Trend<-7% → Q3 强制
                                         否则 → 保持 MCI 象限
                                                    │
每日 HS300 ──→ R20 计算  ──┬──→ 正常仓位（每日调仓）──→ 五资产权重
  （20日收益率）           │      hs300/csi500/cyb/gold/bond
                           │      Q1=90.9% 股票
                           ├──→ L1/L2/L3 触发 ─(覆盖 QW)──→ EM 仓位
                           │
                           └──→ 谷值退出（R20>谷值+10%）──→ 退出 EM
```

### MCI（宏观景气指数）

MCI = PMI_norm × 0.5 + M2_norm × 0.5

- PMI_norm = clamp((PMI - 47) / 6, 0, 1)
- M2_norm = clamp((M2 - 6) / 9, 0, 1)
- 更新频率：月末 PMI 发布 → 月中 M2 发布可能触发切换

| MCI 值 | 象限 | 含义 | 股票权重 |
|:---:|:---:|:---|:---:|
| ≥ 0.53 | Q1（积极） | 宏观景气，全仓进攻 | 90.9% |
| 0.47 ~ 0.53 | Q2（过渡） | 宏观模糊，防御增强 | 68.2% |
| ≤ 0.47 | Q3（防御） | 宏观收缩，保守配置 | 55.0% |

### MA60 趋势覆盖（v2.7 核心新增）

在 MCI 象限基础上叠加沪深 300 的 MA60 偏离度信号：

Trend = P / MA60 - 1

| Trend | 效果 | 说明 |
|:---|:---|:---|
| > +7% | 强制 Q1 | MCI 为 Q2/Q3 时升级 |
| < -7% | 强制 Q3 | MCI 为 Q1/Q2 时降级 |
| 在 ±7% 内 | 保持 MCI | 不干预 |

**触发统计**：+7% 上穿 340 天（11.0%），-7% 下穿 109 天（3.5%），覆盖 MCI 163 天（5.3%）。

### EMERGENCY 三级保护

基于沪深 300 的 20 日收益率（R20）：

| 级别 | 触发 | 配置 |
|:---:|:---|:---|
| L3 | R20 < -20% | HS300 5% + 黄金 70% + 债券 5% + 现金 20% |
| L2 | R20 < -12% | HS300 10% + 黄金 60% + 债券 10% + 现金 20% |
| L1 | R20 < -8% | HS300 15% + 黄金 30% + 债券 25% + 现金 30% |

退出条件（谷值相对退出）：记录谷值 R20，当 R20 > 谷值 + 10% 时退出。

> **优先级**：EM > MA60 覆盖 > MCI 象限。EM 触发时，MA60 和 MCI 均不介入。

### 科技替换机制

从 2019 年起，拆分部分创业板权重为国证科技指数（399303）；从 2022-11 起，国证科技仓位进一步拆分为 50% 国证科技 + 50% 科创芯片 ETF。

## 局限性与已知偏差

- **每日调仓成本假设**：0.3%/次，实际 A 股 ETF 买卖成本 0.3-0.5%/次
- **MA60 滞后**：趋势反转时约 8-10 天滞后（2015 年 -12.1pp 代价）
- **科技替换使用指数而非 ETF**：略高估约 0.1pp/年
- **Q1 债券归零**：适合能承受一定波动的投资者
