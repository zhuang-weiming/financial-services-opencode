## 1. 发布摘要

V21 是从 V20 时代研究中**挑选并冻结**的发布版。初始包含 3 因子
`lv + wt + rev_1m`，后因引擎时序修复发现 `rev_1m` 在正确时序下不产生正
alpha，最终配方为 `lv(0.15) + wt_mom(0.85)`。

## 2. 核心策略

V21 是 2 因子模型（去掉 `rev_1m`），所有因子在 OB 过滤 + 流动性过滤后的
valid pool 内先 `rank(pct=True, na_option='keep')` 归一到 (0,1) 截面排序
（非全 A 股截面），再加权求和。

```
V21 score = lv × 0.15 + wt_mom × 0.85
                ↑            ↑
            低波异常    LazyBear WT
            12 个月     振荡器水平
            滚动 σ(r)
```

| 因子 | 经济学含义 | 实现 |
|------|------------|------|
| `lv` | A 股长期存在低波动率异常 | `rets.rolling(12).std()`，取负 |
| `wt_mom` | 自适应动量（LazyBear WT 周度振荡器） | `wt1_monthly` (HDF5) |

> **`rev_1m` 移除说明**：V21 最初包含 `rev_1m`（20% 权重），但旧引擎的 1 个月
> 信号滞后无意中把反转信号变成了类动量信号，虚高了其表现。

## 5. 重要风险与限制

**发版前必读**：

1. **DSR = 1.0 是极端赢家但要警惕多重检验**：
   - Bootstrap p > 0 为 1.000 — 真实信号
   - DSR=1.0 表明 V21 是 5 试验多重检验下的极端赢家

2. **2008-2009 不在样本**：数据自 2010 起，GFC 时期未覆盖。

3. **数据质量局限**：
   - 2010-2014 市值缺失 ~57%（baostock 无法覆盖的已退市/未上市股）
   - 2024 仍有 434/36720 价格 NaN（约 1.2%），全部是已退市/暂停上市股
   - 2025+ 市值通过价格推导估计（Dec2024 参考市值 × 价格比），非直接摘取

4. **换手率较高**：平均 0.73 = 每月 73% 换手 ≈ 7 只股票。机构实盘建议加
   软换手惩罚。
